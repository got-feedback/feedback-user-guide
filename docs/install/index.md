# Install FeedBack Desktop

Short answer: download the installer for your platform from the latest FeedBack Desktop release, install it, and launch the app.

## Before You Start

- Use the latest release unless a tester or maintainer asks you to use a specific build.
- First launch may take longer than later launches because models and caches may initialize.

## Steps

1. Open the FeedBack Desktop releases page.
2. Download the installer for your platform.
3. Run the installer or app package.
4. Launch FeedBack.
5. Complete the first-run setup.

## Platform Notes

| Platform | Expected Package | Notes |
|---|---|---|
| Windows | `.exe` installer | Windows SmartScreen may warn on first run. |
| macOS | `.dmg` | Use the build matching your Mac architecture. |
| Linux | `.AppImage` or `.deb` | AppImage may need executable permission. |

## What You Should See

- FeedBack opens to first-run setup or the library.
- The app can reach its bundled resources and plugins.
