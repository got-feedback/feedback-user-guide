# What FeedBack Is

FeedBack is an interactive music practice app. It loads song files, displays playable arrangements, listens to your instrument when note detection is enabled, and helps you practice with plugins such as tuner, tab view, fretboard view, metronome, lyrics, stems, and playlists.

## Core Ideas

- A song can have one or more arrangements, such as lead guitar, rhythm guitar, bass, drums, keys, or vocals.
- Plugins add views and tools around the player.
- The library is the collection of songs FeedBack has scanned or imported.
- A feedpak is the portable song package format used by FeedBack.

## What FeedBack Is Not

- It is not a normal media player.
- It is not only a tuner or tab viewer.
- It is not limited to one instrument path.
