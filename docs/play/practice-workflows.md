# Practice Workflows

Short answer: use FeedBack tools together to make practice repeatable: tune, pick a song or playlist, slow down hard sections, loop, and review results.

## Common Workflows

### Learn a New Song

1. Listen through once.
2. Choose the easiest arrangement or section.
3. Slow down if needed.
4. Loop a hard phrase.
5. Increase speed after clean repetitions.

### Maintain a Set

1. Build a playlist.
2. Play all in order.
3. Review missed sections.
4. Mark or tag songs that need work.

### Technique Practice

1. Use a focused plugin or exercise.
2. Use the metronome.
3. Keep sessions short and repeatable.
