# Song Preview

Song Preview lets you quickly hear a song from the library without fully entering the player.

## Use It For

- Auditioning songs before playing.
- Checking a song's audio.
- Browsing a large library faster.

## Basic Workflow

1. Open the library.
2. Use the preview control on a song card where available.
3. Stop preview before opening a different song if needed.

## Common Problems

| Problem | Try This |
|---|---|
| Preview is silent | Confirm the song has playable audio and output volume is up. |
| Preview starts at the wrong place | Check plugin settings if start time is configurable. |
| Preview overlaps playback | Stop preview before launching the player. |
