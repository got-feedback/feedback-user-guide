# Audio Devices

Short answer: FeedBack can listen through microphones, USB instrument cables, audio interfaces, and desktop audio bridge inputs.

## Device Types

| Device Type | Notes |
|---|---|
| Microphone | Easy to start, more room noise and bleed. |
| USB instrument cable | Direct signal, simple for guitar/bass. |
| Audio interface | Best control; choose the correct input channel. |
| Virtual device | Useful for routing, but can be confusing if the OS default changes. |

## Driver Labels

On Windows, one physical interface may appear through multiple driver systems. If you see labels such as ASIO or Windows Audio, choose the one that matches your setup.

On macOS, devices usually appear through CoreAudio.
