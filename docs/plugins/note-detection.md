# Note Detection

Note Detection lets FeedBack listen to your instrument and compare what you play against the active arrangement.

## Use It For

- Scoring.
- Accuracy feedback.
- Practice progress.
- Step-based practice modes where supported.

## Basic Workflow

1. Choose the correct audio input.
2. Run calibration.
3. Open a supported song arrangement.
4. Play while FeedBack listens.
5. Review results.

## Troubleshooting

| Symptom | Try This |
|---|---|
| No scoring | Confirm note detection is enabled and calibrated. |
| Wrong notes detected | Check tuning, input, and calibration. |
| Latency feels wrong | Re-run calibration. |
