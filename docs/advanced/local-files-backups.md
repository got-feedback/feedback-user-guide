# Local Files and Backups

Short answer: keep backups of song files and local data before major updates, hand edits, or bulk library changes.

## What To Back Up

- Song files and feedpak folders.
- Locally edited song metadata.
- User data such as favorites, tags, notes, or practice history where supported.
- Plugin settings if you rely on custom setup.

## Before Hand-Editing Songs

1. Make a copy of the feedpak.
2. Edit the copy.
3. Validate the copy.
4. Import or rescan.

## Before Updating

- Check release notes.
- Back up custom song files.
- Avoid updating in the middle of a test pass unless asked.
