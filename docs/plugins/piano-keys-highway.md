# Piano and Keys Highway

Piano and Keys Highway displays keyboard arrangements in a piano-oriented view.

## Use It For

- Playing keys arrangements.
- Following note timing across a keyboard layout.
- Practicing with MIDI keyboard input where supported.

## Basic Workflow

1. Open a song with a keys arrangement.
2. Open Piano or Keys Highway.
3. Confirm MIDI/input settings if needed.
4. Start playback.

## Common Problems

| Problem | Try This |
|---|---|
| No keys arrangement appears | Choose a song that includes keys. |
| MIDI keyboard does not respond | Check device connection and plugin settings. |
| Notes look too dense | Slow down or practice shorter sections. |
