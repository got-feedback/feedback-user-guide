# Studio

Studio is a recording and multitrack-oriented workflow for FeedBack.

## Use It For

- Recording practice takes where supported.
- Working with multiple tracks.
- Reviewing audio alongside song playback.

## Before You Start

- Confirm audio input and output setup.
- Use headphones when recording with a microphone.
- Keep levels conservative until routing is verified.

## Basic Workflow

1. Open Studio.
2. Select the input source.
3. Arm or prepare a track if the UI requires it.
4. Start playback or recording.
5. Review the take.

## Common Problems

| Problem | Try This |
|---|---|
| Recording is silent | Check input source, channel, and permissions. |
| Audio feeds back | Use headphones and reduce monitoring. |
| Timing feels off | Check latency and calibration settings. |
