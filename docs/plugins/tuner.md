# Tuner Plugin

The Tuner plugin provides a real-time guitar and bass tuner inside FeedBack.

## Use It For

- Tune before playing.
- Tune to the current song.
- Use common guitar and bass presets.
- Create custom tunings.

## How To Open It

Use the Tuner button in the interface or player controls.

## Basic Workflow

1. Open the tuner.
2. Confirm the selected input.
3. Choose Current Song or a preset.
4. Pluck a string.
5. Tune until the indicator shows in tune.

## Settings

| Setting | Meaning |
|---|---|
| Audio input | Device the tuner listens to. |
| Channel | Mono, left, right, or an interface channel where supported. |
| Visualization | Changes the tuner display style. |
| Custom tunings | Adds user-defined tuning presets. |

## Troubleshooting

| Symptom | Try This |
|---|---|
| No movement | Check audio input and channel. |
| Wrong string selected | Use manual string selection or pluck more clearly. |
| Auto-open is distracting | Check plugin settings. |
