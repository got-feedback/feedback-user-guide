# Find More

Find More helps locate related songs or more songs by the same artist.

## Use It For

- Discovering songs related to the current library item.
- Finding more songs by an artist.
- Filling gaps in a practice set.

## Basic Workflow

1. Open a song card action menu.
2. Choose Find More or a similar related-songs action.
3. Review available results.
4. Add or import the item through the supported path.

## Notes

Availability depends on the configured source and build. The guide should not promise that every result can be downloaded directly.

## Common Problems

| Problem | Try This |
|---|---|
| No results | Check spelling, artist metadata, and connection. |
| Wrong artist | Correct song metadata or search manually. |
| Result cannot be imported | Use the import workflow supported by your build. |
