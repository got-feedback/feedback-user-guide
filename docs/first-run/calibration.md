# Calibration

Short answer: calibration helps FeedBack confirm your input and measure timing for note detection.

## When To Use This

- First setup.
- Note detection is late, early, or inconsistent.
- You changed input devices.
- A tester or maintainer asks you to rerun calibration.

## Steps

1. Open calibration from setup or settings.
2. Confirm the selected audio input.
3. Complete the signal check.
4. Complete the string or note check.
5. Complete the latency/timing step if shown.
6. Save the calibration.

## What You Should See

- FeedBack should show activity when you play.
- Each calibration step should move forward after it receives the expected signal.

## Common Problems

| Problem | Try This |
|---|---|
| Latency test does not load | Restart FeedBack and confirm the input is working in the tuner. |
| String names look duplicated | Capture a screenshot and include the selected instrument preset. |
| No input signal | See [Audio Input Problems](../troubleshooting/audio-input.md). |
