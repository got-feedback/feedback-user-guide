# Guitar Theory

Guitar Theory tools help explore scales, chords, intervals, tunings, and fretboard relationships.

## Use It For

- Learning note locations.
- Practicing scale shapes.
- Understanding chord construction.
- Connecting theory to the fretboard.

## Basic Workflow

1. Open Guitar Theory.
2. Choose a topic such as scale, chord, interval, or tuning.
3. Select a key or root note.
4. Use the fretboard display to explore positions.
5. Apply the concept to a song or practice exercise.

## Practice Ideas

- Find the same note in several positions.
- Map a scale across two adjacent positions.
- Compare chord tones against a song's harmony.
