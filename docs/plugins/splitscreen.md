# Splitscreen

Splitscreen shows more than one view at the same time where supported.

## Use It For

- Combining highway and tab views.
- Showing player and reference views together.
- Practicing with multiple visual cues.

## Basic Workflow

1. Open a song.
2. Enable Splitscreen.
3. Choose the views or arrangements to show.
4. Start playback.
5. Adjust layout if controls are available.

## Things To Watch

- Some plugins may not fit well in a split layout.
- Tuning prompts and calibration flows may assume a single active arrangement.
- Smaller screens may need a simpler layout.

## Common Problems

| Problem | Try This |
|---|---|
| View is cramped | Use fewer panels or a larger display. |
| Wrong part appears | Confirm selected arrangement or panel source. |
| A plugin misbehaves in split view | Test it in single-view mode and report both cases. |
