# What Is a feedpak?

Short answer: a feedpak is a portable song package used by FeedBack. It can contain song metadata, arrangements, audio stems, lyrics, timing, cover art, and related files.

## Beginner Version

If you just want to play, think of a feedpak as a FeedBack song file.

## What It Can Contain

- `manifest.yaml` with title, artist, album, and file pointers.
- Arrangement files for playable parts.
- Audio stems such as full mix, vocals, drums, bass, guitar, or other.
- Lyrics and vocal timing.
- Song timeline data such as sections, beats, tempo, and time signatures.
- Cover art.

## Directory or Zip

A feedpak may exist as a folder for editing or as a zip-style package for sharing.
