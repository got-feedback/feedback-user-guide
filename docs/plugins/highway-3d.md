# Highway 3D

Highway 3D shows notes moving toward the player in a 3D lane-style view.

## Use It For

- Playing from a game-style note highway.
- Reading timing and note density visually.
- Practicing arrangements where the highway is easier than tab.

## Basic Workflow

1. Open a song arrangement.
2. Choose the 3D highway view.
3. Confirm tuning and arrangement.
4. Press Play.

## Common Problems

| Problem | Try This |
|---|---|
| Notes appear on unexpected strings | Confirm tuning and arrangement. |
| Performance is poor | Try another view or reduce other active panels. |
| Scoring is wrong | Check audio input and calibration. |
