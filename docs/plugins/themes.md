# Themes

Themes change the visual appearance of FeedBack.

## Use It For

- Adjusting the app's look.
- Improving contrast or comfort.
- Matching screenshots or streaming layouts.

## Basic Workflow

1. Open Settings or the Themes plugin.
2. Pick a theme.
3. Confirm text, cards, and controls remain readable.
4. Keep the theme or switch back.

## Accessibility Notes

Choose a theme with enough contrast for extended practice. If a theme makes controls hard to read, switch back before reporting unrelated UI issues.

## Common Problems

| Problem | Try This |
|---|---|
| Text is hard to read | Switch themes and report the theme name. |
| Theme does not apply everywhere | Restart FeedBack and confirm the plugin is enabled. |
