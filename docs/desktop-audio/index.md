# Desktop Audio

Short answer: FeedBack Desktop includes native audio features that go beyond the browser app, including device selection, low-latency input paths, amp modeling, and advanced routing work.

## Common Topics

- Audio input devices.
- ASIO, WASAPI, and CoreAudio.
- VST or amp modeling support.
- Backing track playback.
- Streamer/extra output routing.
