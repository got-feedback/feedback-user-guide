# Playback Problems

Short answer: playback problems usually involve the selected song, audio output, queue state, or plugin interaction.

## Symptoms

- Song does not start.
- Song starts but returns to the menu.
- Queue stops after one song.
- Backing track is silent.
- Player view loads but controls do not respond.

## First Checks

1. Try a different song.
2. Restart FeedBack.
3. Confirm audio output works outside FeedBack.
4. Disable non-essential plugins and retry.
5. If using a playlist or setlist, try playing one song directly.

## Queue Problems

If Play all stops after one song, include whether you started from a playlist, setlist, album, or filtered library. Also include whether a results screen appeared.

## Reporting Details

- Song filename or title.
- Arrangement.
- Whether it happens with one song or all songs.
- Whether it happens from direct play or queue/play-all.
- Diagnostics bundle.
