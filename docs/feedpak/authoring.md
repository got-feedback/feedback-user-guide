# Authoring and Editing feedpak Files

Short answer: advanced users can author or edit feedpak files by changing the manifest and related data files.

## Before You Start

- Back up the file first.
- Use the official feedpak spec when editing by hand.
- Validate after editing.

## Common Authoring Tasks

- Correct title, artist, album, or year.
- Add lyrics.
- Add or update arrangements.
- Add stems.
- Add cover art.
