# Tab View

Tab View shows playable guitar or bass tablature during song playback.

## Use It For

- Reading fretted notes in tab form.
- Following a part without using a 3D highway.
- Checking string and fret positions.

## Basic Workflow

1. Open a song with a guitar or bass arrangement.
2. Enable or open Tab View.
3. Start playback.
4. Follow the scrolling tab while playing.

## What To Check

- The selected arrangement.
- The song tuning.
- Whether the tab view is synced to playback.

## Common Problems

| Problem | Try This |
|---|---|
| Tab does not match what you expect | Confirm the selected arrangement and tuning. |
| Tab view does not appear | Confirm the plugin is installed and enabled. |
| Timing feels off | Check playback latency and calibration if scoring is also involved. |
