# Common Terms

Use this page when a word in the app or docs is unfamiliar.

| Term | Meaning |
|---|---|
| FeedBack | The app and project. |
| feedpak | A song package/file format used by FeedBack. |
| song | A playable item in your library. |
| chart | A specific authored version of a song. |
| arrangement | A playable part inside a chart, such as lead guitar, bass, drums, or keys. |
| plugin | A feature module that adds a view or tool. |
| stem | An isolated audio track such as vocals, drums, bass, guitar, or other. |
| library | Your scanned/imported song collection. |
| scan | Search the songs folder for new or changed song files. |
| rescan | Re-read song files and metadata. |
| calibration | Setup that helps FeedBack measure input and timing. |
| mastery | Practice status derived from your accuracy/history. |
