# Practice

The Practice plugin tracks practice activity and helps turn song work into repeatable sessions.

## Use It For

- Reviewing recent practice.
- Tracking time spent in songs or sections.
- Finding songs that need more work.
- Building a routine around saved lists or tags.

## Basic Workflow

1. Open the Practice plugin or practice dashboard.
2. Review recent songs or sessions.
3. Pick a song that needs work.
4. Practice a focused section.
5. Review result and repeat later.

## Good Practice Data

Practice tracking is only useful if the selected song, arrangement, input, and scoring path are correct. If results look wrong, fix setup before relying on the data.

## Common Problems

| Problem | Try This |
|---|---|
| Practice history looks empty | Play a scored song or confirm the plugin is enabled. |
| Accuracy seems wrong | Check audio input, tuning, and calibration. |
| A song appears under an old name | Rescan metadata or check local edits. |
