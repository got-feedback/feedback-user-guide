# Metronome

The Metronome plugin provides a click or visual beat reference for practice.

## Use It For

- Practicing timing.
- Warming up before a song.
- Working through difficult rhythmic passages slowly.
- Keeping steady tempo when no backing audio is needed.

## Basic Workflow

1. Open the Metronome plugin.
2. Choose a tempo or follow the song tempo where supported.
3. Start the click.
4. Practice the phrase or section.
5. Stop the click before normal playback if it is not needed.

## Settings To Expect

| Setting | Meaning |
|---|---|
| Tempo | Beats per minute. |
| Meter | Beat grouping such as 4/4 or 6/8 where supported. |
| Volume | Click level. |
| Visual flash | Beat cue without relying on sound. |

## Common Problems

| Problem | Try This |
|---|---|
| Click is too loud | Lower metronome volume before starting playback. |
| Tempo does not match song | Check whether the song has tempo metadata. |
