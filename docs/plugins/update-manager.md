# Update Manager

The Update Manager installs, updates, and removes plugins where supported by your FeedBack build.

## Use It For

- Installing optional plugins.
- Updating installed plugins.
- Removing plugins you no longer use.
- Checking plugin status.

## Basic Workflow

1. Open Settings.
2. Open Plugins or Update Manager.
3. Review available updates.
4. Install or update the plugin.
5. Restart FeedBack if prompted.

## Common Problems

| Problem | Try This |
|---|---|
| Plugin does not appear after install | Restart FeedBack. |
| Update fails | Check internet access and try again. |
| Plugin causes problems | Disable it and report the issue with diagnostics. |
