# Rig Builder

Rig Builder is for building or inspecting tone rigs, signal chains, and related tone information where supported.

## Use It For

- Viewing a song's intended tone setup.
- Building an amp/pedal/cab style chain.
- Pairing tone data with a song or arrangement.

## Basic Workflow

1. Open Rig Builder.
2. Choose a rig or create a new one.
3. Add or adjust blocks.
4. Test the tone at low volume.
5. Save or apply the rig where supported.

## Common Problems

| Problem | Try This |
|---|---|
| Rig does not make sound | Check audio input, output, and tone engine support. |
| Block is unsupported | Use a supported realization or fallback tone. |
| Tone is too loud | Lower output before enabling effects. |
