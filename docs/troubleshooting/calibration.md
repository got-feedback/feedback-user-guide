# Calibration Problems

Short answer: calibration needs a working input signal, the correct instrument setup, and a responsive detection path.

## Symptoms

- Signal step does not move.
- Latency test does not load.
- String names look duplicated.
- Calibration finishes but scoring feels wrong.

## Fixes To Try

1. Test the tuner first. If the tuner does not move, solve audio input before calibration.
2. Re-select your input device and channel.
3. Confirm instrument and string count.
4. Restart FeedBack and rerun calibration.
5. Export diagnostics if the problem repeats.

## Reporting Details

Include:

- Which calibration step failed.
- Instrument preset.
- Screenshot if visual labels are wrong.
- Whether normal note detection works outside calibration.
