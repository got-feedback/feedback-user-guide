# Plugin Problems

Short answer: plugin problems can come from installation state, incompatible song data, missing permissions, or a plugin-specific bug.

## Symptoms

- Plugin button is missing.
- Plugin opens but shows no data.
- Plugin causes errors.
- Plugin works in one song but not another.

## First Checks

1. Confirm the plugin is installed.
2. Confirm it is enabled.
3. Restart FeedBack.
4. Try a song that should support the plugin.
5. Check whether the plugin has required setup, such as input or stems.

## Song Compatibility

Some plugins need specific data:

| Plugin Type | Common Requirement |
|---|---|
| Tab/fretboard views | Fretted arrangement data. |
| Lyrics | Lyric files. |
| Stem tools | Audio stems. |
| Note detection | Calibrated input and supported arrangement. |

## Reporting Details

- Plugin name.
- Song and arrangement.
- What you clicked.
- Screenshot.
- Diagnostics bundle.
