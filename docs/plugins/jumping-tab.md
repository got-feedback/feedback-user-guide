# Jumping Tab

Jumping Tab is a horizontal tab-style view with motion cues for following a fretted arrangement.

## Use It For

- Practicing from tab with a more animated visual style.
- Seeing upcoming notes as a horizontal path.
- Using movement cues instead of a 3D highway.

## Basic Workflow

1. Open a guitar or bass arrangement.
2. Open Jumping Tab.
3. Start playback.
4. Follow the moving tab cues while playing.

## Common Problems

| Problem | Try This |
|---|---|
| Notes do not match your instrument | Confirm the selected arrangement and tuning. |
| View feels too busy | Try Tab View or Fretboard View instead. |
| Plugin is missing | Confirm it is installed and enabled. |
