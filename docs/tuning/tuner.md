# Tuner

Short answer: the tuner listens to your input and helps tune your guitar or bass to a preset or the current song.

## When To Use This

- Before playing a song.
- When the current song uses a different tuning.
- When you changed instruments.
- When note detection seems wrong.

## Steps

1. Open the Tuner button or tuner panel.
2. Confirm the input device and channel if shown.
3. Choose Current Song, a preset, or a custom tuning.
4. Pluck one string at a time.
5. Tune until the indicator is centered/in tune.
6. Close or leave the tuner open depending on your workflow.

## Current Song Tuning

When opened from the player, the tuner can follow the tuning required by the current song arrangement.

## Auto-Open Behavior

Some builds can prompt or auto-open the tuner when the next song needs a different tuning. If this is distracting, check tuner or plugin settings for auto-open options.

## Common Problems

| Problem | Try This |
|---|---|
| Tuner does not move | Check the selected audio input and channel. |
| Tuner opens then closes | Check auto-play and tuner auto-open settings; see troubleshooting. |
| Tuner shows the wrong note | Pluck one string clearly and reduce background noise. |
| Song tuning looks wrong | Confirm the selected arrangement. |
