# No Signal

Short answer: if FeedBack shows no input signal, solve that before tuning, calibration, or scoring.

## First Checks

1. Confirm the instrument, cable, mic, or interface is plugged in.
2. Confirm the interface gain is up.
3. Confirm the input is not muted.
4. Confirm the operating system can see the device.
5. Re-select the input in FeedBack.
6. Try the tuner as the simplest signal test.

## Device-Specific Checks

| Setup | Check |
|---|---|
| USB instrument cable | Make sure it is selected as input, not output. |
| Audio interface | Pick the correct channel. |
| Microphone | Check OS microphone permission. |
| Virtual device | Confirm the source app is sending audio to it. |

## If Signal Appears in the OS but Not FeedBack

- Restart FeedBack.
- Re-select the input device.
- Try another driver path if available.
- Export diagnostics after reproducing the issue.
