# Setlist

Setlist helps build ordered song lists for rehearsal, performance practice, or repeated play-throughs.

## Use It For

- Practicing a fixed group of songs.
- Rehearsing songs in order.
- Separating a gig list from casual library browsing.
- Choosing arrangement-specific practice paths where supported.

## Basic Workflow

1. Open the Setlist plugin or playlist/setlist area.
2. Create a setlist.
3. Add songs in the desired order.
4. Reorder songs if needed.
5. Use Play all or the equivalent start action.

## Setlist vs Playlist

A playlist is a general saved list. A setlist is usually more intentional: it has an order you expect to repeat.

## Common Problems

| Problem | Try This |
|---|---|
| Playback stops after one song | Check whether your build uses the newer queue path; see [Playback Problems](../troubleshooting/playback.md). |
| Wrong arrangement opens | Check per-song arrangement choices if the plugin exposes them. |
| A song is missing | Confirm it still exists in the library. |
