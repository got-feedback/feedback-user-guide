# Fretboard View

Fretboard View shows notes on a virtual fretboard while a song plays.

## Use It For

- Seeing where notes land on the instrument.
- Connecting tab positions to fretboard shapes.
- Practicing position awareness.

## Basic Workflow

1. Open a song with a guitar or bass arrangement.
2. Open Fretboard View.
3. Start playback.
4. Watch active notes and positions on the fretboard.

## Practice Ideas

- Pause on a phrase and name the note locations.
- Compare the fretboard view with Tab View.
- Use the view to identify position shifts.

## Common Problems

| Problem | Try This |
|---|---|
| Fretboard seems to show the wrong tuning | Confirm the selected arrangement and song tuning. |
| View is missing | Confirm the plugin is enabled. |
