# Plugins

Short answer: plugins add tools, views, and workflows to FeedBack.

## Common Plugin Types

- Player views: tab, fretboard, highway, lyrics, drums, keys.
- Practice tools: tuner, metronome, note detection, step mode, practice journal.
- Library tools: setlists, playlists, find more, update manager.
- Audio tools: stems, stem mixer, NAM tone, studio.

## How To Use Plugins

1. Open Settings or the plugin manager.
2. Enable or install the plugin.
3. Restart FeedBack if required.
4. Open the plugin from its button, player area, or library action.
