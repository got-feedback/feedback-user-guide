# Step Mode

Step Mode turns a song or phrase into a stop-and-go practice flow.

## Use It For

- Learning a difficult phrase note by note.
- Practicing slowly without constantly rewinding.
- Waiting for the correct note before moving on where note detection supports it.

## Basic Workflow

1. Open a supported song arrangement.
2. Enable Step Mode.
3. Start the section or phrase.
4. Play the prompted note or advance manually if supported.
5. Repeat until the phrase feels comfortable.

## Practice Tips

- Use Step Mode for short sections first.
- Tune before using it; wrong tuning can make every step feel wrong.
- If note detection is enabled, calibrate before relying on automatic advancement.

## Common Problems

| Problem | Try This |
|---|---|
| Step does not advance | Confirm note detection input, tuning, and calibration. |
| It advances too easily | Check input noise and sensitivity if settings are available. |
| Plugin is missing | Confirm it is installed and enabled. |
