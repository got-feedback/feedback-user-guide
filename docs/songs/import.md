# Import Songs

Short answer: add song files to FeedBack, then scan the library so they appear on the Songs screen.

## Before You Start

- FeedBack song packages are usually feedpak files.
- Desktop users can use the songs folder workflow.
- Remote or browser-hosted setups may need an import/upload path because the browser cannot see the server's folders.

## Steps

1. Open the Songs screen.
2. Use Add Songs, Import, or the songs folder workflow available in your build.
3. Add the song file.
4. Run a scan if the library does not update automatically.
5. Confirm the new song appears in the library.

## Scan vs Import

| Action | Use It When |
|---|---|
| Scan for new songs | You placed files in the songs folder. |
| Import a file | You have a feedpak somewhere else on disk. |
| Full rescan | You edited metadata or need FeedBack to re-read existing files. |

## What You Should See

- New song cards appear in the library.
- If scan progress is visible, it should finish with a completion state.

## Common Problems

| Problem | Try This |
|---|---|
| Song does not appear | Run scan/rescan and check file location. |
| Metadata looks old | Run a full rescan. |
| Duplicate versions appear | See multiple charts and metadata docs when available. |
