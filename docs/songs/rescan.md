# Scan and Rescan

Short answer: scan finds new or changed songs; full rescan re-reads metadata for songs already in the library.

## When To Use This

- You copied new songs into the songs folder.
- A song is missing from the library.
- You edited a song file or metadata.
- The library view looks stale.

## Steps

1. Open the Songs screen.
2. Choose Refresh, Scan, or Rescan depending on the build.
3. Wait for scan progress to finish.
4. Confirm the song list updates.

## Choosing the Right Action

| Action | Meaning |
|---|---|
| Refresh or scan | Look for new/changed files. |
| Full rescan | Re-read metadata from existing files. |
| Import | Copy or add a specific file from outside the songs folder. |

## Common Problems

| Problem | Try This |
|---|---|
| New song still missing | Confirm the file is in the songs folder and is a supported song package. |
| Edited title did not change | Run a full rescan. |
| Scan seems stuck | Export diagnostics and include the current file shown, if visible. |
