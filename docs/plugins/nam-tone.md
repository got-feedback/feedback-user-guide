# NAM Tone

NAM Tone uses amp modeling and cabinet/IR style processing to create guitar or bass tone inside FeedBack where supported.

## Use It For

- Practicing with an in-app guitar tone.
- Trying song-specific tones.
- Routing a dry instrument input through amp modeling.
- Stream or recording setups where an in-app tone is useful.

## Before You Start

- Use a clean direct input if possible.
- Confirm the correct input device and channel.
- Keep output volume conservative until the tone path is verified.

## Basic Workflow

1. Select the correct audio input.
2. Open the NAM Tone plugin.
3. Choose or load a tone.
4. Play quietly and confirm signal.
5. Adjust input and output levels.

## Common Problems

| Problem | Try This |
|---|---|
| No tone | Confirm input signal and plugin enable state. |
| Tone is noisy or distorted | Lower input gain and check the model/IR. |
| Latency is distracting | Use a lower-latency driver path where available. |
