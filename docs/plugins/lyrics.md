# Lyrics

Lyrics plugins display or create timed lyrics for songs.

## Plugin Types

| Plugin | Purpose |
|---|---|
| Lyrics Sync | Align plain text lyrics to audio. |
| Lyrics Karaoke | Show timed lyrics and vocal pitch information where available. |

## Use It For

- Singing along.
- Checking lyric timing.
- Creating timed lyric data for a song.
- Building karaoke-style views.

## Basic Workflow

1. Open a song with lyrics or vocals.
2. Open the lyrics plugin.
3. Choose the available lyric track if more than one exists.
4. Start playback.

## Authoring Notes

Advanced lyric alignment may require vocals audio and a backend alignment service. Some songs may include translated or transliterated lyric tracks.

## Common Problems

| Problem | Try This |
|---|---|
| No lyrics appear | The song may not include lyric data. |
| Lyrics are out of sync | Check whether lyrics were aligned for this recording. |
| Alignment takes a long time | First-time model setup or server processing may be running. |
