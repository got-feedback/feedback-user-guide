# Drum Highway

Drum Highway displays drum parts in a lane-based view.

## Use It For

- Playing drum arrangements.
- Following rhythm lanes.
- Practicing with MIDI drum pads where supported.

## Basic Workflow

1. Open a song with a drum arrangement.
2. Open Drum Highway.
3. Confirm input setup if using drum pads.
4. Start playback.

## Common Problems

| Problem | Try This |
|---|---|
| Drum arrangement is missing | Choose a song that includes drums. |
| Pads do not trigger | Check MIDI/input settings for the plugin. |
| View does not appear | Confirm the plugin is installed and enabled. |
